# Arcane Void

Educational-recreational project to teach myself using SDL on Racket.


## Installation

1. [Racket](https://racket-lang.org/)
2. `raco pkg install sdl3`
3. Brew/apt/yum: `sdl3 sdl3_image sdl3_ttf`.

## Running

```sh
./run.sh
```

or Emacs: `M-x package-install racket-mode`, `C-x C-f arcanevoid.rkt`, `C-c C-c` or `racket-run`.
